position

static(default), relative, absolute, sticky, fixed

공통 사항 : static을 제외한 나머지 값들은 z-index라는 공간 값을 갖는다.

absolute : 기준점(기본적으로 body가 기준점)을 기준으로 top은 top으로부터, left는 left로부터, right는 right로 부터, bottom은 bottom으로부터 내부로 이동한다.(음수 일 때 바깥쪽으로 이동) 방향 값을 지정하지 않으면 원래 static이었던 자리에 있으려고 한다. - 가로 방향 left와 right 두가지 값이 있을 때는 left가 우선, top과 bottom 두가지 값이 있을 때는 top이 우선 absolute 쓰는 경우 1.대부분 겹쳐져 사용하는 경우 2.배너같은 움직이는 요소 - 다른요소와 관계없이 움직임 3.static이었던 상태로는 절대 위치할 수 없는 경우 (더보기 같은 버튼 - html상 내용 끝, 마지막에 있는데 css로 위치를 위로 변경해야함) 기준점 변경 가능 조상 요소 중에서 static이 아닌 가장 가까운 요소가 기준이 된다.

relative: absolute와 비슷한 개념, 투명한 static과 눈에 보이는(붕 떠있는) absolute가 동시에 작동하는 방식

absolute는 기준 body로부터 움직임, relative는 기준이 원래 자기가 있던 자리

단락

fixed: 브라우저를 기준으로 고정되는 포지션 값

fixed는 브라우저에 고정되는 값을 의미한다.

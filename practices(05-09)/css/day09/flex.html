flex

flex 는 종(열), 횡(행) 정렬을 하는 속성과 값이다.

flex 는 기본값이 행 정렬로, 부모요소(container)와 컨텐츠(contents)간의 속성과 값의 협력관계가 이루어져야 한다.
항상 부모요소에서 diplay:flex;라고 속성과 값을 적용한 다음 방향과 위치 값을 지정한다.
이후 컨텐츠에서 flex값으로 기준을 중심으로 두고 얼만큼 커질 것인지 작아질 것인지 적용한다.

flex-direction: row(default), column;
flex-wrap: nowrap(default)(1행 정렬), wrap(여러행 정렬)
justify-content: flex-start(default), space-between(처음과 끝에 첫 번째 마지막이 각각 위치하고 나머지는 똑같은 여백을 갖는다), space-around(모든 자식들의 좌우 여백이 똑같이 갖는다), center, flex-end;
align-content: stretch(default)(처음부터 끝까지 균등하게 늘어난다), flex-start, space-between, space-around, center, flex-end; 2행 이상일 때 세로정렬 -> 마진과 패딩이 합쳐져서 크기가 더해지니 주의
align-items: stretch(default)(처음부터 끝까지 균등하게 늘어난다), flex-start, center, flex-end; 1행일 때 세로정렬 align-self: flex-start, center, flex-end; 각각 값을 가짐

flex:0 1 auto; 기본 값 (flex-grow flex-shrink flex-basis)
flex-grow는 container의 여유공간(빈공간)을 중심으로 숫자의 비율만큼 각 flex요소들이 나눠 갖는다. flex-shrink는 container가 각 flex를 합친 것보다 더 작아질 때 그 크기만큼 flex요소들이 숫자의 비율만큼 나눠서 없앤다. flex-basis(기준점-가장 중요), flex-grow flex-shrink로 커지거나 작아질 수 있는 기준점(auto는 컨텐츠만큼의 크기)

flex요소의 margin:auto값은 container값의 빈공간의 값을 자동으로 갖게 된다.
